Masterize Swift.

Esse curso cobre os fundamentos do desenvolvimento de aplicativos iOS na linguagem de programação Swift. Você aprenderá a usar ferramentas de desenvolvimento como XCode, design de interfaces e interações, avaliar sua usabilidade e integrar câmera, foto e informações de localização para aprimorar seu aplicativo. No projeto final, você aplicará suas habilidades para criar um aplicativo de edição de fotos totalmente funcional para iPhone, iPad e Apple Watch. É necessário um computador Mac para ter sucesso neste curso.
