Explore ser front-end dev.

Se você curte programação e design, esse é o lugar ideal para você! Explore o seu lado lógico e criativo, produzindo aplicações incríveis e funcionais!

Nosso foco aqui é a criação de páginas web, que você tenha a base necessária para estudar os frameworks mais quentes. Por isso, vamos nos aprofundar ainda mais em HTML e CSS e principalmente no JavaScript já que essa é a linguagem base da web. Antes de continuar seus estudos, lembre-se que é importante ter noção de lógica de programação e git.

O que eu vou estudar?

■ HTML e CSS #1 - Sua primeira página web.

■ HTML e CSS #2 - Posicionamento, flexbox e grid.

■ HTML e CSS #3 - Forms e tables

■ Javascript #1 - Primeiros passos.

■ Javascript #2 - Funções, arrays e objetos.

■ Javascript #3 - Arrow functions.

■ Javascript #4 - Orientação a objetos.

■ Front-UI #1 - Responsividade.

■ Front-UI #2 - Descomplicando o CSS.

■ Front-UI #3 - Layout.
